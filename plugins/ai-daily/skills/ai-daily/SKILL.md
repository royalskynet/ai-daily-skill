# Global Daily - 全球熱門資訊日報

透過 Brave Search API 獲取全球最新熱門新聞，利用 Claude AI 智能摘要、分類並翻譯成中文，生成結構化的 Markdown 日報或是精美響應式網頁。

## 快速開始

```bash
# 查詢今天的熱門資訊 (預設)
今天全球資訊

# 查詢特定日期的資訊
2026-04-06的熱門新聞

# 生成網頁
今天熱門資訊，生成網頁

# 生成分享圖片
為今日熱報生成分享圖片
```

## 支援的查詢類型

| 查詢類型 | 示例 | 描述 |
|------|----------|-------------|
| **相對日期** | "今天熱門資訊" "昨天的全球動態" | 查詢最近 24-48 小時內的新聞 |
| **絕對日期** | "2026-04-06的新聞" | 指定特定 YYYY-MM-DD 日期 |
| **網頁生成** | "生成網頁" "製作HTML頁面" | 將日報轉為精美網頁 |
| **圖片生成** | "生成分享卡片" "製作日報卡片" | 生成適合 IG/社交媒體的分享圖 |

## 工作流 (Workflow)

複製此清單以跟踪進度:

```
進度:
- [ ] 步驟 1: 解析用戶要求的目標日期
- [ ] 步驟 2: 通過 Brave Search API 獲取新聞 (過去 24 小時)
- [ ] 步驟 3: 使用 Claude 進行新聞價值分析與分類
- [ ] 步驟 4: 生成結構化的 Markdown 輸出
- [ ] 步驟 5: (選填) 根據要求生成網頁主題
- [ ] 步驟 6: (選填) 生成 4:5 比例的分享圖片
```

---

## 步驟 2: 獲取資料

運行抓取腳本以獲取資料：

```bash
python plugins/ai-daily/skills/ai-daily/scripts/fetch_news.py --relative today
```

這會調用 Brave Search API 抓取趨向熱點（由 `.env` 中的 `BRAVE_SEARCH_QUERY` 決定）。

---

## 步驟 3: 提取與分析 (Claude Analyzer)

使用內置 Claude AI 的分析能力：

1. **提取內容**：從 Brave 獲取的 snippets 中提取關鍵資訊。
2. **核心摘要**：生成 3-5 條今日最重要的要點（每條 < 50 字）。
3. **分類歸納**：
   - 國際要聞 (world)
   - 經濟財經 (finance)
   - 科技創新 (tech)
   - 社會生活 (lifestyle)
   - 體育競技 (sports)
   - 娛樂影視 (entertainment)
4. **關鍵詞提取**：抓取當天的熱門公司、人物與事件。

---

## 步驟 5: 網頁生成 (選填)

**僅在用戶明確要求時觸發**。可選主題：
- **蘋果風** (Apple Style) - 極簡、低調奢華
- **深海藍** (Deep Sea) - 商務、權威感
- **秋日暖陽** (Autumn) - 溫馨、充滿活力

網頁會儲存於 `docs/` 目錄下。

---

## 步驟 6: 分享圖片生成 (選填)

**觸發條件**：用戶提到「生成分享圖」、「做成卡片」或「生成圖片」。

1. 壓縮當前日報內容。
2. 調用 Firefly Card API 生成 1080x1350 (4:5) 的 PNG 圖片。
3. 儲存於 `docs/images/`。

---

## 配置 (Configuration)

在環境變數中確保以下 key 已填寫：
- `BRAVE_API_KEY`: 必要，用於新聞抓取。
- `CLAUDE_API_KEY`: 必要，用於分析內容。
- `BRAVE_SEARCH_QUERY`: 選填，預設為 `"top trending global news"`。
